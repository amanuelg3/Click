If you are running an application with the Dojo remote runtime project, you must be connected to the Internet. If your application uses Dojo widgets, your project EGL build path must point to either of the following Dojo projects shipped with EDT: org.eclipse.edt.rui.dojo or org.eclipse.edt.rui.dojo.remote. For more information about using Dojo and EGL, see the following topic in the help system: "Preparing to use the EGL Dojo widgets."
